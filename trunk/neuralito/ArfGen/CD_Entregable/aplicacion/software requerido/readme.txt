Instaladores para windows X86 del software requerido para el funcionamiento de la aplicación SurfForecaster:

	- Java RunTime 1.6
	- MYSQL Server 5.1
	- Apache Tomcat 6.0

Si desea utilizar la aplicación en otra plataforma sirvase descargarse las versiones correspondientes en:
	
	- Java RunTime: 	http://java.sun.com/javase/downloads/index.jsp
	- Mysql: 			http://dev.mysql.com/downloads/mysql/
	- Apache Tomcat: 	http://tomcat.apache.org/download-60.cgi
	
Una vez instalado el software correspondiente siga las instrucciones de instalación ubicadas en "aplicacion/bin/instalacion.txt"
